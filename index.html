How do you develop your website?

1
Plan

Who's the target audience? What type of business and website? Which web stack will be used? What about cross-compatabilty?

2
Design

I start designing the website based on what was planned on the beginning.

3
Execute

I start writing code for the website based on the design. What I design is what I build.

4
Test

I check for bugs, cross-compatabilty, speed, and UI/UX design.

5
Deploy

I push my code into github and deploy the website. When working with other developers, I instead will push and send a pull request.

6
Repeat

I rinse and repeat the process as I add more features and content on the website.
